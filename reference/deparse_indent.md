# Deparse an expression and indent for pretty-printing

Deparse an expression and indent for pretty-printing

## Usage

``` r
deparse_indent(x, indent = 0L)
```

## Arguments

- x:

  A `code` object

- indent:

  An `integer` number of spaces or a string to prefix each line of the
  deparsed output.

## Value

An indented version of the deparsed string from `x`.
