# Determine the number of source code lines of a given [srcref](https://rdrr.io/r/base/srcfile.html)

Determine the number of source code lines of a given
[srcref](https://rdrr.io/r/base/srcfile.html)

## Usage

``` r
srcref_nlines(x)
```

## Arguments

- x:

  A [srcref](https://rdrr.io/r/base/srcfile.html) object

## Value

The number of lines in the original source of a
[srcref](https://rdrr.io/r/base/srcfile.html)
