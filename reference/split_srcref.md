# Split a Source Reference at specific lines

Split a Source Reference at specific lines

## Usage

``` r
split_srcref(sr, where)
```

## Arguments

- sr:

  An original [srcref](https://rdrr.io/r/base/srcfile.html) object

- where:

  A numeric vector of line numbers where the
  [srcref](https://rdrr.io/r/base/srcfile.html) should be split

## Value

A list of [srcref](https://rdrr.io/r/base/srcfile.html)
